  1. Programming Career: Simple Needs for Programming Basics

    A lot of way to learn programming language. In following outline, I tell about basic needs before start learn programming to who never been used computer.

    1. Connecting Network

      Before you learn programming language, you need to connect network to your computer. Only then you can use relevant application for programming on your computer. The ways of connect network are shown follow.

      1. Connecting Network with Wi-Fi

        2. If you have wi-fi, you can connect wi-fi to your computer. I will explain step by step how to connect Wi-fi.

        Connecting to Wi-Fi network using Taskbar

        1. Click the Network icon in the bottom-right corner of the taskbar.
        2. Select the wireless network to connect.
        3. (Optional) Check the Connect automatically option.
        4. Click the Connect button.

        5. Confirm the network security key (password).

        6. Click the Next button.
        7. Confirm whether the device should be discoverable in the network by other PCs.
        If you do not find network icon, there have another way.

        Connecting to Wi-Fi network using Setting

        1. Open Settings.
        2. Click on Network & Security.
        3. Click on Wi-Fi.
        4. Click the Manage known networks option.

        5. Click the Add a new network button.

        6. Confirm the name of the network.
        7. Use the drop-down menu to select the Security type – For example, WPA2-Personal AES.
        8. Confirm the network security key (password).

        9. Check the Connect automatically option.
        10. Check the Connect even if this network is not broadcasting option (if required).
        11. Click the Save button.
        Once you complete the steps, the laptop will connect to the network using the Wi-Fi connection.

      2. Connecting Network with Phone

        3. If you don’t have Wi-fi, you can connect network from your phone to computer. Connecting network with phone require one more step than connecting with computer as shown below.
        1. On your phone, open the Settings app and then tap Personal Hotspot.
        2. Just below the slider button you will see password for your smartphone’s hotspot. If you want to change password tap it.
        3. The phone automatically uses the device name as the hotpot name.
        4. Open the computer and connect network with your device name (connecting is the same like wi-fi connecting).

    2. Learning English is Importing Part of a Programming Basics

      3. English is not essential but it is important for learning programming basics. Most documentation, references and anything at all related to programming will be in English. Many of 4 the languages themselves use English terminology and a huge amount of the documentation, tutorials, videos, etc... are in English. So, if you do not know English, you are cutting yourself off from a huge amount of learning resources. The keywords of many programming languages are in English, so not knowing English would make it slightly harder to learn. It would be harder to get sample code and answers to questions without knowing English. Not only to learn programming, but also to access the knowledge written in English, you should learn English.
