Programming Career: Simple Needs for Programming Basics

A lot of ways to learn programming language. In the following outline, I explain about basic needs to people who have never used computers before they start to learn programming.

  1. Connecting to a Network

    Before you learn a programming language, you need to connect your computer to a network. Only then, you can use a relevant application for programming on your computer. The Ways of connecting to a network are shown as follow.

    1. Connecting to a Wi-Fi Network

      2. If you have Wi-Fi, you can connect Wi-Fi to your computer. I explain step by step how to connect Wi-Fi.

      Connecting to a Wi-Fi network using Taskbar

      1. Click the Network icon in the bottom-right corner of the taskbar.
      2. Select a wireless network to connect.
      3. (Optional) Check the Connect automatically option.
      4. Click the Connect button.

      5. Confirm the network security key (password).

      6. Click the Next button.
      7. Confirm whether the device should be discoverable in the network by other PCs.
      If you do not find network icon, there is another way.

      Connecting to a Wi-Fi network using Settings

      1. Open Settings.
      2. Click on Network & Security.
      3. Click on Wi-Fi.
      4. Click the Manage known networks option.

      5. Click the Add a new network button.

      6. Confirm the name of the network.
      7. Use the drop-down menu to select the Security type – For example, WPA2-Personal AES.
      8. Confirm the network security key (password).

      9. Check the Connect automatically option.
      10. Check the Connect even if this network is not broadcasting option (if required).
      11. Click the Save button.
      Once you complete the steps, the laptop will connect to the network using the Wi-Fi connection.

    2. Connecting Network with a Smartphone

      3. If you don’t have Wi-fi, you can connect your computer to a network through your phone. Connecting to a network via a phone requires the following steps before connecting a computer as shown below.
      1. On your phone, open the Settings app and then tap Personal Hotspot.
      2. Just below the slider button you will see password for your smartphone’s hotspot. If you want to change password tap it.
      3. The phone automatically uses the device name as the hotspot name.
      4. Open the computer and connect network with your device name (connecting is by the same way like Wi-Fi connection).

  2. Learning English is Important Part of Programming Basics

    3. English is not essential but it is important for learning programming basics. Most documents, references and anything at all related to programming are written in English. Many of the languages themselves use English terminology ,a huge amount of the documentation, tutorials, videos, etc... are in English. So, if you do not know English, you miss a huge amount of learning resources. The keywords of many programming languages are in English. Thus, not knowing English would make it harder to learn. It would be harder to get sample code and answers to questions without knowing English. Not only to learn programming, but also to access the knowledge written in English, you should learn English.
